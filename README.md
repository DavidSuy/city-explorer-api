# City Explorer

**Author**: David Suy
**Version**: 1.0.0

## Overview

City explorer allows you to search for a city. Then you will be given a map, weather, and a list of movies that has the city name in it.

## Getting Started

To get started make .env file, follow .env.sample then install npm

## Architecture
